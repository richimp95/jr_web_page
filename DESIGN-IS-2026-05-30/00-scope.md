# Scope — NRGY Landing Page Design Audit

**Date:** 2026-05-30
**Audited artifact:** `/home/richi/projects/jr_web_page/` — static marketing site
**Live URL:** https://richimp95.github.io/jr_web_page/
**Stack:** Pure HTML/CSS/JS, GitHub Pages (no build step)

## Primary user
Distribuidor / retailer en Centroamérica buscando un nuevo producto de impulso para sus puntos de venta.

## Primary task
Entender qué es NRGY, convencerse de que es un buen producto para distribuir, y enviar una solicitud de distribución.

## Screens audited
- `index.html` — única página (single-page)
- `style.css` — 805 líneas
- `main.js` — 109 líneas

## Constraints
- Brand: Midnight Navy #1F3042, Copper #C68662, Paper #FAF7F4
- Fonts: Varela Round (display) + Hanken Grotesk (body)
- No backend — formulario vía Formspree
- GitHub Pages — archivos estáticos únicamente
- Bilingual ES/EN via data attributes + localStorage

## Out of scope
- SEO técnico (robots.txt, sitemap)
- Performance benchmarking en producción real
- Backend / formulario processing
